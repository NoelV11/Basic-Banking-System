## What does it do?
This project mimics the functionality offered by any normal banking website, allowing a account holder, to:-

Transact Money

View Transaction Records and

View Customer Information

## Tech Stack Used
It is powered using the following:-
- Bootstrap
- CSS
- HTML
- JQuery and 
- JavaScript
